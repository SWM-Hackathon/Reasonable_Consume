## 💸 '착한 소비'는 어떤 서비스인가요?
- 소비자들이 쉽게 원하는 착한가격업소를 찾을 수 있고,
- 인기있는 착한가격업소는 홍보되는,
- 하나의 착한 소비 문화를 만드는 서비스입니다.
- 소비자들은 손쉽게 착한업소를 찾게 되고,
- 업소 입장에서는 착한업소가 되기 위해 양질의 서비스를 제공하게 되고, 부가적으로 국가 지원도 받는,
- 모두가 윈윈하는 경제 활성화 및 물가 안정의 선순환 고리를 형성할 수 있는 서비스입니다.
